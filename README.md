# zookeeper-demo

`icu.sunnyc.zk.demo1` 使用 zk 原生 API 实现服务器动态上下线案例
`icu.sunnyc.zk.demo2` 使用 zk 原生 API 实现分布式锁案例
`icu.sunnyc.zk.demo3` 使用 curator 框架的分布式锁案例
